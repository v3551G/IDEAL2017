Paper Submission

Authors are invited to submit their manuscripts (in pdf format) written in English by the deadline via the conference online submission system (accessible via IDEAL 2017 website). All submissions will be refereed by experts in the field based on originality, significance, quality and clarity. All contributions must be original, should not have been published elsewhere and must not be submitted elsewhere during the review period.

Papers should not exceed 8 pages and must comply with the format of Springer LNCS/LNAI Proceedings. (see https://www.springer.com/computer/lncs?SGWID=0-164-6-793341-0)

Submission to Workshop/Special Sessions using one of the following methods:
* Please submit using the IDEAL 2017 submission link as above as to the main track but make note of the paper ID after the submission. Then email the paper ID and Special Session name to ideal2017@easychair.org
* Add the Workshop or Special Session name to the first line of the abstract when submitting using the IDEAL 2017 submission link.

Submission System:   https://easychair.org/conferences/?conf=ideal2017
